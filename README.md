# ADHD Life Planner

An ADHD-friendly web planner designed to help users manage spending, routines, and daily life organization in a way that feels simpler and less overwhelming.

## Purpose
This project was created to turn everyday chaos into a clear, usable system. It is designed with ADHD-friendly structure in mind, focusing on visibility, motivation, and reducing friction.

## Features
- Spending and budget planning
- Routines and life organization
- Interactive planner interface
- Structured workflow design
- Persistent state for ongoing use

## Built With
- HTML
- CSS
- JavaScript

## Why I Built It
I wanted to create a tool that helps real people manage real life in a way that feels supportive instead of overwhelming. This project reflects my interest in front-end development, product thinking, and building tools that solve practical problems.
